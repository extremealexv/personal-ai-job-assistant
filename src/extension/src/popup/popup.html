Help

How to use:

  1. Navigate to a job application page (Workday, Greenhouse, Lever, or Taleo)
  2. Click the "Fill Application" button
  3. Review the filled information
  4. Submit the application manually (or enable auto-submit)

Supported Platforms:

  • Workday (*.myworkdayjobs.com)
  • Greenhouse (boards.greenhouse.io)
  • Lever (jobs.lever.co)
  • Taleo (*.taleo.net)

Troubleshooting:

If autofill doesn't work:

  • Refresh the page and try again
  • Make sure you're logged in
  • Check that the platform is enabled in settings
  • Report issues on GitHub
